# Dashboard Comercial com Power BI e PostgreSQL

Este projeto apresenta um dashboard comercial desenvolvido no Power BI a partir de uma base educacional utilizada em curso de SQL da Hitadore/Udemy.

O objetivo foi construir uma análise comercial completa, passando por conexão com banco PostgreSQL, modelagem de dados, criação de tabela calendário, medidas DAX, análise de funil, desempenho por lojas, produtos e clientes.

## Objetivo do projeto

Analisar dados comerciais de uma operação fictícia, identificando:

- evolução da receita mensal;
- desempenho por loja;
- desempenho por marca e modelo de veículo;
- comportamento do funil de conversão;
- distribuição de clientes por estado e cidade.

## Ferramentas utilizadas

- PostgreSQL
- Power BI Desktop
- Power Query
- DAX
- Photoshop
- GitHub

## Páginas do relatório

### 1. Visão Geral Comercial

Página executiva com os principais indicadores do projeto:

- Total de vendas;
- Receita;
- Ticket médio;
- Taxa de conversão;
- Total de visitas;
- Evolução da receita mensal;
- Top lojas por vendas.

### 2. Lojas

Análise do desempenho das lojas por receita, vendas e ticket médio.

### 3. Produtos

Análise por marca, modelo, ano e valor de venda dos veículos.

### 4. Funil de Conversão

Análise da jornada comercial:

- Visitas;
- Carrinhos;
- Checkout iniciado;
- Checkout finalizado;
- Vendas.

O principal gargalo identificado está entre carrinho e início do checkout.

### 5. Clientes

Análise do perfil dos clientes por localização, com foco em receita e concentração por estado/cidade.

## Principais medidas DAX

Foram criadas medidas para:

- Total de visitas;
- Total de vendas;
- Receita;
- Ticket médio;
- Taxa de conversão;
- Taxas do funil;
- Clientes únicos.

## Fonte dos dados

Base de dados educacional utilizada em curso de SQL da Hitadore/Udemy.
